### ASCII Mirrors

Reads a text file and mirrors the text on both sides. Main package in ASCII Mirror/task/src

Ex. input:
```
            ^__^             
    _______/(oo)     
/\/(       /(__) 
   | w----||     
   ||     ||     
```

Ex. output:
```
            ^__^ | ^__^            
    _______/(oo) | (oo)\_______    
/\/(       /(__) | (__)\       )\/\
   | w----||     |     ||----w |   
   ||     ||     |     ||     ||   
```
